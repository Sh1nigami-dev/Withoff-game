# Withoff-game
Данный программный продукт является результатом курсовой работы по дисциплине «Технология разработки программного обеспечения», выполненной студентом группы 24ИСП-2 Макаровым М. А..
## Описание проекта
АС «Игра Витхоффа» — это компьютерная реализация классической математической игры для двух игроков (человек и компьютер). Основная цель проекта — программно смоделировать безошибочную выигрышную стратегию компьютера, основанную на математическом аппарате теории игр.
## Правила игры
Игра ведется с использованием двух кучек фишек. За один ход игрок может:
1. Забрать любое количество фишек из одной кучки.
2. Забрать одинаковое количество фишек из обеих кучек одновременно.
Побеждает тот, кто забирает последнюю фишку, приводя игру в состояние (0, 0).
## Ключевые возможности
- Искусственный интеллект: Алгоритм компьютера рассчитывает оптимальные ходы на основе формулы золотого сечения (формулы Бетти), стремясь перевести игру в «холодные» (проигрышные для оппонента) позиции.
- Гибкая настройка: Возможность задать начальное количество фишек в кучках и выбрать, кто сделает первый ход (игрок или ПК).
- Валидация данных: Система блокирует некорректный ввод (буквы, отрицательные числа, нулевые ходы), обеспечивая стабильность работы.
- Интуитивный интерфейс: Лаконичный WIMP-интерфейс, разделенный на блоки инициализации и игрового процесса.
## Технологический стек
- Язык программирования: C#.
- Платформа: .NET Framework.
- Среда разработки: Visual Studio 2019.
- Методология проектирования: IDEF0 (построено в Ramus Educational).
## Системные требования
- ОС: Windows 7/8/10/11.
- Процессор: тактовая частота от 1,2 ГГц.
- Оперативная память: не менее 512 МБ.
- Разрешение экрана: не менее 1024 x 768 пикселей.
- ПО: установлен .NET Runtime.
## Управление
- Мышь: Основное взаимодействие с кнопками и переключателями.
- Tab / Shift+Tab: Переход между полями ввода.
- Enter: Подтверждение хода или начала игры.
- Alt + F4: Быстрое закрытие программы.
